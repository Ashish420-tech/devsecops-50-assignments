# Security Incident Response Runbook

## 1. Detection
- Alert received via PagerDuty (simulated)
- Notification sent to Slack

## 2. Analysis
- Review logs
- Identify suspicious activity
- Determine severity

## 3. Containment
- Block malicious IPs
- Restrict access

## 4. Eradication
- Remove threats
- Patch vulnerabilities

## 5. Recovery
- Restore services
- Monitor system

## 6. Communication
- Slack updates
- Jira ticket tracking

## 7. Closure
- Mark incident resolved
- Document learnings
