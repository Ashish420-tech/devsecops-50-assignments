# Assignment 22 - Jenkins Pipeline Security Hardening

## Objective
Implement a secure Jenkins pipeline using Jenkins Credentials Store and avoid hardcoded secrets.

## Tools Used
- Jenkins
- Jenkins Pipeline
- Jenkins Credentials Store

## Architecture
Developer → Jenkins Pipeline → Secure Credential Handling

## Implementation Steps
1. Created Jenkins pipeline
2. Demonstrated insecure pipeline with hardcoded secret
3. Stored secret in Jenkins Credentials Store
4. Updated pipeline to use secure credentials

## Key Learnings
- Avoid hardcoding secrets in pipelines
- Use Jenkins Credentials Store
- Secure handling of sensitive data in CI/CD
