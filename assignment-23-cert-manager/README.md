# 🔐 TLS Certificate Management with cert-manager

## 📌 Objective
Automate TLS certificate issuance using cert-manager and Let's Encrypt.

## 🛠 Tools Used
- Kubernetes
- cert-manager
- Let's Encrypt
- NGINX Ingress

## 🚀 Steps
1. Install cert-manager
2. Create ClusterIssuer
3. Deploy application
4. Configure Ingress with TLS
5. Verify certificate

## 🔐 Outcome
- Automated SSL provisioning
- Secure HTTPS communication
- Zero manual certificate handling

## 📸 Screenshots
- Pods running
- Certificate issued
- HTTPS working
