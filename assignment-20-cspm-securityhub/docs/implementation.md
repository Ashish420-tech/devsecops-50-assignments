# Implementation - CSPM with AWS Security Hub

---

## Step 1: Enable AWS Security Hub

AWS Security Hub is enabled from the AWS Management Console.
It acts as a central service to aggregate security findings.

---

## Step 2: Enable GuardDuty

GuardDuty is enabled to monitor AWS accounts for suspicious activities and threats.

---

## Step 3: Enable AWS Config

AWS Config is enabled to track resource configurations and evaluate compliance rules.

---

## Step 4: Review Security Findings

All findings from GuardDuty and AWS Config are aggregated in AWS Security Hub.
Users can review and analyze security issues from a centralized dashboard.

---

## Outcome

* Centralized visibility of security findings
* Detection of potential threats
* Basic compliance monitoring
